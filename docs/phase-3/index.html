Phase 3: The Albu Process

Building a reproducible, semi-automated semiconductor fabrication line from the process validated in Phase 2.

Process Naming Convention

The Albu process is named by its minimum reliable feature size. The name is earned through characterization, not aspiration — it reflects what the process can actually deliver with acceptable yield.

Process NameMinimum FeatureMeaning
Albu 25M25μm25-micron minimum feature, stable and reproducible
Albu 10M10μm10-micron minimum feature, stable and reproducible
Albu 5M5μm5-micron minimum feature (stretch goal)

The “M” stands for microns. Getting a named, characterized process is the goal of Phase 3 — a process where you can state the feature size, yield, and device parameters with confidence because they have been measured across dozens of devices.

What Needs Automating

Phase 2 validates the transistor process with manual methods. Phase 3 turns those manual steps into a repeatable, semi-automated fabrication line. Every source of human variability needs to be reduced or eliminated.

The Lithography Station

The core of the fab line is a maskless direct-write lithography system — a precision XY scanning stage that focuses a single-mode laser through a microscope objective to write patterns directly onto photoresist-coated substrates. A beam splitter enables through-the-lens (TTL) camera alignment, so the same optical path used for exposure is also used for finding alignment marks.

Core Components

  • Laser source: Single-mode 405nm diode (or 275nm UV-C LED for higher resolution). Focused through microscope objective to a diffraction-limited spot.
  • Microscope objective: 20x or 50x long-working-distance (LWD) objective. Defines the exposure spot size and field of view.
  • Beam splitter: 50/50 plate or cube splitter. Sends laser down through the objective and reflects the sample image up to the camera.
  • Machine vision camera: 5MP+ monochrome camera on the tube lens port. Provides live view of the substrate for alignment and focus.
  • Precision XY stage: Closed-loop linear stages with optical encoders. ~1μm positioning accuracy. The substrate moves; the optics stay fixed.
  • Z focus stage: Motorized or piezo-driven for autofocus. Maintains the laser spot at the resist surface across the entire die.
  • Motion controller: Klipper firmware running on a BTT Octopus board with TMC2209 stepper drivers.
  • Software: Python + OpenCV for alignment mark detection, GDS-to-scan-path conversion, and exposure sequencing.

Why Maskless

Maskless direct-write eliminates the need to produce a physical photomask for each layer. Pattern changes are instant (modify the GDS file and re-expose), and layer-to-layer alignment is computed from detected alignment marks rather than mechanical pin registration. The pipeline: GDS file → scan coordinates → affine transform → exposure.

Exposure Strategy: Step-and-Scan

The objective field of view limits how much area can be exposed at once. The lithography station uses a step-and-scan strategy: expose one field, step to the next, repeat until the full die is covered. At 10μm features with a 20x objective, a 7mm die takes approximately 3–4 minutes per layer.

Backup approach: If direct-write resolution proves insufficient, a 10x photomask reduction system (mask printed at 10x scale, projected through a reduction objective) provides an alternative path to 10μm features.

Yield Characterization

Before attempting a CPU, we need statistical data on the process. Gut feeling and a handful of working transistors are not enough — we need numbers.

Methodology

  • Fabricate 50+ identical transistors across multiple substrates
  • Measure Vth, on/off ratio, and mobility for each device
  • Calculate mean, standard deviation, and yield (% of devices that meet spec)
  • Track defect density per unit area

Yield Targets

Target CPUTransistor CountRequired Per-Transistor YieldEstimated Die Yield
Albu-4 (4-bit CPU)~1,000>99%~90%
SERV (RISC-V core)~4,500>99.9%~95%

Getting from 99% to 99.9% per-transistor yield is the central engineering challenge of Phase 3. Every defect source must be identified and eliminated: particles in the resist, pinholes in the gate dielectric, incomplete etching, contact resistance variation, film thickness non-uniformity.

This data determines Phase 4. The yield numbers from Phase 3 directly dictate whether we attempt the Albu-4 custom CPU (~1,000 transistors) or the SERV RISC-V core (~4,500 transistors). There is no point designing a CPU until we know what the process can deliver.

Critical Validation Tests

The lithography station and automated process line require these tests before attempting multi-layer IC fabrication.

Test 3.1: Stage Accuracy Verification

MeasureMethodRequirement
Positioning accuracyLaser-mark a grid of dots at known 100μm spacing, measure under microscope<5μm error
RepeatabilityCommand stage to same position 20 times, measure spread<2μm standard deviation
BacklashApproach same position from +X and −X, measure difference<3μm

Test 3.2: Alignment Mark Detection

VariableSweep
Mark size10, 20, 50, 100μm cross marks
Objective magnification10x, 20x, 50x
Detection algorithmTemplate matching, edge detection, centroid fitting

Measure: Detected position vs known position across 50 marks. Need <1μm RMS error for 10μm feature alignment.

Test 3.3: Focused Laser Spot Size

VariableSweep
ObjectiveRepurposed BEBANG 40x, new 10x, new 20x
Laser sourceGenmitsu 5W multimode, single-mode 405nm diode

Method: Ablate single-pass lines in thin aluminum film on glass. Measure line width under microscope = effective spot size. The single-mode diode through a 20x objective should achieve 3–5μm; the multimode 5W diode will be 10–20μm.

Test 3.4: Multi-Layer Alignment Accuracy

The ultimate integration test. Write alignment marks + a test pattern on layer 1 (resist or ablation). Process the layer. Write layer 2 using alignment marks for registration. Measure the overlay error between layers.

Pass criterion: <3μm overlay error (1/3 of minimum feature size) across a 7mm die area.
